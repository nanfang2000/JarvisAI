## 📱 响应式设计

### 断点设置
```css
/* 最小宽度要求 */
--min-width: 1000px;
--min-height: 700px;

/* 自适应断点 */
--breakpoint-sm: 1200px;  /* 小屏模式 */
--breakpoint-md: 1400px;  /* 中屏模式 */
--breakpoint-lg: 1600px;  /* 大屏模式 */
```

### 布局适配
- **小屏(1000-1200px)**: Avatar面板35%, Content面板65%
- **中屏(1200-1400px)**: Avatar面板40%, Content面板60% (默认)
- **大屏(1400px+)**: Avatar面板35%, Content面板65%, 增加侧边栏

## 🔧 组件状态设计

### Avatar状态
- `idle`: 等待状态 - 😊 静态表情
- `listening`: 聆听状态 - 👂 动态音波
- `thinking`: 思考状态 - 🤔 进度条动画
- `speaking`: 回复状态 - 💬 嘴型同步
- `error`: 错误状态 - 😟 错误提示

### 连接状态
- `connected`: 🟢 绿色指示器 + "JARVIS在线"
- `connecting`: 🟡 黄色动画 + "连接中..."
- `disconnected`: 🔴 红色指示器 + "连接中断"
- `error`: ❌ 错误图标 + "连接失败"

### 交互反馈
- **按钮悬停**: 0.15s过渡 + 颜色加深
- **点击反馈**: 0.1s缩放动画
- **加载状态**: 旋转动画 + 进度指示
- **消息发送**: 滑入动画 + 类型指示器

这个设计规范将指导整个JARVIS系统的界面开发，确保一致性和用户体验的科幻感！
